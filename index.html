Feel free to use the material linked here for yourself or your team. If you decide to use it, it would be great if you can drop us a short message on how you are using the material and give us feedback. Please use the following mail address: EmbeddedTesterWorkshop@gmail.com

These workshops would include more time for the exercises and additional topics like Test Reporting or how to use git. Send us a message if you are interested: EmbeddedTesterWorkshop@gmail.com
