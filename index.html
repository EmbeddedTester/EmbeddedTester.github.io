Deployment to Heroku

If you want to deploy to Heroku, you need to do the following steps:

  • Fork the Github repo for pairing
  • Create an account on Heroku
  • Create an app on Heroku
  • On the deploy tab in Heroku, select Github and authorize Heroku to access your Github account
  • Setup automatic deployment, every push to your Github fork will then be automatically deployed to Heroku
