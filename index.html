Setup Postman

You can use Postman to make service calls:

  • Install Postman
  • Start Postm and and select Import (next to New button)
  • Select import by URL and enter: http://fixturefinder-service.herokuapp.com/v2/api-docs
  • You will now have a collection with different service calls
  • Right click on the collection and select "Edit"
  • Select the tab "Variables" and update baseUrl to https://fixturefinder-service.herokuapp.com/

Deployment to Heroku

If you want to deploy to Heroku, you need to do the following steps:

  • Fork the Github repo, see above for the different links for each setup
  • Create an account on Heroku
  • Create an app on Heroku
  • On the deploy tab in Heroku, select Github and authorize Heroku to access your Github account
  • Setup automatic deployment, every push to your Github fork will then be automatically deployed to Heroku
